# 实时市场分析系统 - 使用指南

基于 **akshare** 的股票/加密货币数据源集成

## 📦 安装依赖

```bash
pip install akshare pandas
```

## 🚀 快速开始

### 1. 基础配置

在 `~/.nanobot/config.json` 中启用市场监控工具（已自动注册，无需额外配置）：

```json
{
  "agents": {
    "defaults": {
      "provider": "openrouter",
      "model": "anthropic/claude-opus-4-6"
    }
  },
  "channels": {
    "telegram": {
      "enabled": true,
      "token": "${TELEGRAM_BOT_TOKEN}",
      "allowFrom": ["YOUR_USER_ID"]
    }
  }
}
```

### 2. 启动 nanobot

```bash
nanobot gateway
```

## ⏰ 定时任务配置

### 每小时价格检查

在 `~/.nanobot/config.json` 中添加：

```json
{
  "cron": {
    "enabled": true,
    "tasks": [
      {
        "name": "hourly_btc_check",
        "schedule": "0 * * * *",
        "command": "crypto_price symbol=\"BTC\""
      }
    ]
  }
}
```

### 每日市场报告

```json
{
  "cron": {
    "tasks": [
      {
        "name": "daily_market_report",
        "schedule": "0 9 * * 1-5",
        "command": "帮我生成今日A股市场日报，包括：上证指数、深证成指、创业板指的最新价格和涨跌幅"
      }
    ]
  }
}
```

### 异常波动预警

创建自定义技能 `skills/market-alert/SKILL.md`：

```markdown
# Market Alert Skill

## Description
监控股票/加密货币价格异常波动，超过阈值时发送预警。

## Usage
当检测到以下情况时触发：
- 单日涨跌幅超过 5%
- 成交量异常放大（超过平均值 2 倍）
- 突破关键支撑/阻力位

## Examples
- 监控贵州茅台，如果涨跌幅超过 3%，发送预警
- 监控 BTC，如果价格跌破 $60,000，立即通知
```

## ⚠️ 注意事项

1. **数据延迟**: akshare 数据可能有 1-5 分钟延迟，不适合高频交易
2. **API 限制**: 频繁调用可能被限流，建议使用缓存（已内置 60 秒缓存）
3. **市场时间**: A股交易时间为 9:30-11:30, 13:00-15:00
4. **数据准确性**: 仅供学习研究，不构成投资建议

## 🛠️ 故障排查

### 问题 1: 找不到股票代码

```
错误: 未找到股票代码: XXXX
```

**解决**:
- 确认代码格式正确（A股 6 位数字，不含市场后缀）
- 检查是否在交易时间内
- 尝试使用 web_search 工具搜索正确的代码

### 问题 2: akshare 导入失败

```
错误: akshare not installed
```

**解决**:
```bash
pip install akshare pandas
```

### 问题 3: 数据获取超时

```
错误: Request timeout
```

**解决**:
- 检查网络连接
- 尝试更换网络环境
- akshare 依赖多个数据源，某些源可能暂时不可用
